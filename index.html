Meus Projetos

Projeto 1: Sistema de livrarias

Um sistema completo de livraria online para a Livraria Lelet, integrando login social, gestão de livros e fluxo de compra, com foco em design moderno, responsivo e intuitivo. A aplicação permite autenticação tradicional e via Google, cadastro e gerenciamento completo de livros (criação, edição, exclusão e busca), além de carrinho de compras com cálculo automático de valores e finalização simulada. O projeto prioriza experiência do usuário, organização das funcionalidades e uma interface limpa e interativa, utilizando Supabase como banco de dados link do projeto.

Projeto 2: SteamNow

Desenvolvi um projeto de otimização de banco de dados no PostgreSQL para a plataforma fictícia StreamNow, aplicando particionamento por LIST (país) e RANGE (data) para melhorar desempenho, organização e escalabilidade. Estruturei tabelas particionadas e implementei consultas analíticas estratégicas, demonstrando boas práticas em modelagem, performance e preparação do sistema para alto volume de dados link do projeto.

Projeto 3: Site sobre Zigbee

Este projeto é uma página web informativa sobre a tecnologia Zigbee. A página apresenta de forma clara e visualmente atraente os principais conceitos sobre a tecnologia Zigbee, um protocolo de comunicação sem fio amplamente utilizado na Internet das Coisas (IoT) link do projeto.

Projeto 4: Banco de dados locadora de veiculos

Desenvolvi um sistema completo de banco de dados para locadoras de veículos que automatiza e integra todas as operações do negócio: cadastro de clientes com CPF único, gestão da frota com disponibilidade em tempo real, controle de locações com cálculo automático de valores, registro de transações financeiras e histórico de manutenções. A solução elimina planilhas manuais, reduz erros operacionais e oferece visibilidade completa do negócio para uma gestão mais eficiente e assertiva link do projeto.

Projeto 5: NutriAI

Desenvolvi junto a uma equipe uma solução tecnológica inteligente que melhora a qualidade de vida e os resultados do tratamento de pacientes com câncer de estômago e diabetes, através de um plano alimentar 100% personalizado e dinâmico, utilizando inteligência artificial para adaptar as recomendações nutricionais em tempo real link do prejeto.

Projeto 6: Recomendador de receitas fitness

Este projeto é uma automação criada no n8n. O fluxo utiliza embeddings da OpenAI combinados com um banco vetorial no Supabase para pesquisar receitas fitness por similaridade semântica. O usuário envia uma palavra-chave → o workflow converte para embedding → o Supabase retorna as receitas mais próximas → o n8n entrega a resposta link do projeto.
